# Math_156_Group_Project

## Dataset installation
The dataset is available on kaggle at https://www.kaggle.com/datasets/tolgadincer/labeled-chest-xray-images
You should place your dataset inside a directory called 'dataset'
Format 'Math_156_Group_Project/dataset/chest_xray/...'

## Dependencies
Run the following in conda to install the needed libraries
conda env create -f environment.yml
